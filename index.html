SVAMITVA QGIS Plugin

This QGIS plugin generates property cards and final deliverables under the SVAMITVA project. It automates:

  • Property card generation
  • Village Habitation Traverse Map
  • Correlation Map
  • Property Parcel Map
  • Grama Kantam Map (with old boundary)
